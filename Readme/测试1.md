# 测试1 — 传感器线性度校准记录

日期：2026-05-03

## 标定系数

`DEFAULT_SCALE_FACTOR = 0.00000267f`

## 三点 / 四点验证

| 砝码 | 显示值 | 误差 | 误差率 |
|------|--------|------|--------|
| 15g | 0.016kg | +1g | +6.7% |
| 213g | 0.194kg | -19g | -8.9% |
| 354g | 0.377kg | +23g | +6.5% |
| 1000g | 1.000kg | 0g | 0% |

## 现象

误差随负载 **先负后正**：213g 处偏低 9%，354g 处偏高 6.5%，1000g 处归零。

## 结论

非系数问题——是称重传感器自身的物理非线性。整体误差范围在 ±25g 以内。

## 处理

暂不处理。生鲜结算场景对 ±25g 误差可接受。如需更高精度，可后续实现分段线性插值校准（多校准点 + 区间线性补偿）。

## 算法版本

- 滑动均值缓冲：深度 4
- 自适应一阶滞后滤波：α_fast=0.5, α_slow=0.1，切换阈值 10g
- 输出死区：2g
- 开机调零：20 样本剪裁均值（排序后去首尾各 3，取中间 14 平均）
