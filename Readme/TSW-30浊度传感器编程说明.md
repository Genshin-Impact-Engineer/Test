# TSW-30 浊度传感器模块 — AI 编程说明

## 基本参数

| 参数 | 值 |
|------|-----|
| 工作电压 | **5V DC**（不可使用 3.3V） |
| 工作电流 | 40mA (MAX) |
| 响应时间 | < 500ms |
| 操作温度 | -20°C ~ 90°C |
| 传感器型号 | TSW-30 |
| 测量范围 | **0~1000 NTU ±30** |
| 红外发射管峰值波长 | 940nm |
| 光敏二极管峰值波长 | 880nm |

## 引脚定义

| 引脚 | 名称 | 说明 |
|------|------|------|
| 1 | VCC | 供电正极，**5V** |
| 2 | **AO** | **模拟量输出** 0~5V → 接 MCU ADC |
| 3 | **DO** | **数字量输出** 高/低电平（可调阈值） |
| 4 | GND | 供电负极 |

> DO 引脚：浊度 < 阈值 → 输出高电平；浊度 > 阈值 → 输出低电平（D1 指示灯亮）

## 核心公式

### 浊度与电压关系（标准曲线）

```
TU = -865.68 × U + K
```

- `TU` = 当前浊度值 (NTU)
- `U` = 当前温度下模块输出电压 (V)
- `K` = 截距值（需标定）

### 温度补偿公式

```
ΔU = -0.0192 × (T - 25)
```

- `ΔU` = 温度变化引起的电压差 (V)
- `T` = 当前测量温度 (°C)

### 校准后的浊度计算公式

```
U₂₅℃ = U_测试 - ΔU                  // 补偿到 25°C 的电压
K    = -865.68 × U₂₅℃              // 计算 K 值（用 0NTU 标定液）
TU   = -865.68 × U_当前 + K          // 最终浊度公式
```

## 校准步骤

1. 准备 **0NTU 标定液**（或纯净水/蒸馏水）
2. 测量当前水温 `T_测试`，模块输出电压 `U_测试`
3. 计算电压补偿：`ΔU = -0.0192 × (T_测试 - 25)`
4. 计算 25°C 标准电压：`U₂₅℃ = U_测试 - ΔU`
5. 计算 K 值：`K = -865.68 × U₂₅℃`
6. 代入最终公式：`TU = -865.68 × U + K`

## 浊度等级划分（参考）

| 等级 | 电压范围 | 说明 |
|------|----------|------|
| 1 级 | 2.96V ~ 5V | 较清澈 |
| 2 级 | 2.64V ~ 2.96V | 轻微浑浊 |
| 3 级 | 1.84V ~ 2.64V | 中等浑浊 |
| 4 级 | 0 ~ 1.84V | 严重浑浊 |

> 阈值可根据实际需求调整。

## ADC 采样参考

```c
// 伪代码示例 (STM32)
#define VREF        3.3     // MCU 参考电压
#define ADC_RES     4096    // 12位 ADC

// 读取 ADC 并转换为电压值
float read_turbidity_voltage(void) {
    uint16_t adc_val = adc_read();
    float voltage = (float)adc_val / ADC_RES * VREF;
    // 注意：模块 AO 输出 0~5V，如果 MCU 是 3.3V 需分压或使用 5V  tolerant ADC
    return voltage;
}

// 计算浊度 (需要先标定得到 K)
float calc_turbidity(float voltage, float K) {
    return -865.68f * voltage + K;
}
```

## 数字模式使用

1. 准备目标浊度的对比液
2. 调节模块上的 **蓝色电位器**，使 D1 指示灯刚好亮起（临界点）
3. DO 引脚接 MCU 的 GPIO（输入模式）
4. DO = 高电平 → 浊度未超标；DO = 低电平 → 浊度超标

## 注意事项

1. **传感器顶端不防水**，严禁完全浸入水中，水面不能没过顶盖与壳体连接处
2. 避免日光或灯光直射传感器，会影响光敏二极管接收（波长范围 500~1050nm），增大测量误差
3. 传感器头部壳体需保持清洁，避免污染物遮挡
4. 测量需做温度补偿，否则精度会受影响
5. 模块 **AO 输出 0~5V**，如果 MCU 是 3.3V 系统，需要用电阻分压或运放调理后再接 ADC
