# 28BYJ48 步进电机 — AI 编程说明

## 基本参数

| 参数 | 值 |
|------|-----|
| 电机型号 | 28BYJ48 |
| 驱动电压 | 5V |
| 相数 | 4 相 5 线 |
| 驱动芯片 | ULN2003 |
| 减速比 | **1:64** |
| 步进角（未减速） | 5.625° |
| 步进角（减速后） | 5.625° / 64 ≈ 0.0879° |
| **转一圈所需脉冲** | **360 / 5.625 × 64 = 4096 个** |
| 电机电流 | 约 200mA |

## 接线定义

| 信号 | 说明 |
|------|------|
| A (蓝色) | 线圈 A 相 → GPIO |
| B (粉色) | 线圈 B 相 → GPIO |
| C (黄色) | 线圈 C 相 → GPIO |
| D (橙色) | 线圈 D 相 → GPIO |
| 公共端 (红色) | **接 5V** |

> ULN2003 驱动板 IN1~IN4 对应电机的 A~D 相。

## 驱动时序

### 四相八拍驱动（推荐，力矩大、振动小）

**正转次序：** A → AB → B → BC → C → CD → D → DA

| 步 | A | B | C | D | 端口值 |
|----|---|---|---|---|--------|
| 1 | 1 | 0 | 0 | 0 | 0x08 |
| 2 | 1 | 1 | 0 | 0 | 0x0C |
| 3 | 0 | 1 | 0 | 0 | 0x04 |
| 4 | 0 | 1 | 1 | 0 | 0x06 |
| 5 | 0 | 0 | 1 | 0 | 0x02 |
| 6 | 0 | 0 | 1 | 1 | 0x03 |
| 7 | 0 | 0 | 0 | 1 | 0x01 |
| 8 | 1 | 0 | 0 | 1 | 0x09 |

**反转：** 将上表倒序执行。

### 四相单四拍驱动（速度较快，力矩较小）

**正转次序：** A → B → C → D

| 步 | A | B | C | D | 端口值 |
|----|---|---|---|---|--------|
| 1 | 1 | 0 | 0 | 0 | 0x08 |
| 2 | 0 | 1 | 0 | 0 | 0x04 |
| 3 | 0 | 0 | 1 | 0 | 0x02 |
| 4 | 0 | 0 | 0 | 1 | 0x01 |

### 四相双四拍驱动（力矩较大）

**正转次序：** AB → BC → CD → DA

| 步 | A | B | C | D | 端口值 |
|----|---|---|---|---|--------|
| 1 | 1 | 1 | 0 | 0 | 0x0C |
| 2 | 0 | 1 | 1 | 0 | 0x06 |
| 3 | 0 | 0 | 1 | 1 | 0x03 |
| 4 | 1 | 0 | 0 | 1 | 0x09 |

## 关键公式

```
转一圈脉冲数 = (360° / 步进角) × 减速比
             = (360 / 5.625) × 64
             = 4096 个脉冲

转速 (rpm) = 脉冲频率(Hz) × 60 / 4096
```

## GPIO 初始化和控制参考

```c
// 假设 A→PA0, B→PA1, C→PA2, D→PA3  (STM32 示例)
// 正转表 (四相八拍)
const uint8_t step_table[8] = {0x08, 0x0C, 0x04, 0x06, 0x02, 0x03, 0x01, 0x09};

// 双四拍正转表 (文档中原始驱动方式)
const uint8_t step_table_double[4] = {0x0C, 0x06, 0x03, 0x09};

// 发送一步
void step_write(uint8_t val) {
    // 将低 4 位写入对应 GPIO 引脚
    GPIOA->ODR = (GPIOA->ODR & 0xFFF0) | (val & 0x0F);
    delay_ms(2);  // 速度控制：调节此延时改变转速
}
```

## 速度-延时对照参考

| 转速目标 | 每步延时 |
|----------|----------|
| 慢速 (~5 rpm) | 约 2~3 ms |
| 中速 (~15 rpm) | 约 1 ms |
| 快速 (~30 rpm) | 约 0.5 ms |

> 注意：启动时不要立即跳到最高速，需要逐步加速，否则可能丢步。

## 注意事项

1. 电机是**减速电机**，减速比 1:64，转一圈需要 4096 个脉冲
2. 无脉冲时电机处于**静止并自锁**状态（有保持力矩）
3. 改变脉冲顺序即可**改变旋转方向**
4. 转速与脉冲频率成正比，频率越高转速越快
5. 可以用 **PWM+定时器中断** 实现精确的步进控制
6. ULN2003 是集电极开路输出，公共端接 5V，输出低电平驱动
