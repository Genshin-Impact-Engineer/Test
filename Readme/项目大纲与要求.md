## 🎛️ 阀门开度控制

### 档位与角度对应关系

工程中将阀门开度分为 **6 个档位（0-5 档）**，每档对应 30°：

| 档位 | 相对零点角度 | 累计步数 | 阀门开度 | 说明 |
|------|-------------|---------|---------|------|
| **0 档** | 0° | 0 步 | 0% | 零点位置/全关 |
| **1 档** | 30° | 350 步 | 20% | 小开 |
| **2 档** | 60° | 700 步 | 40% | 中小开 |
| **3 档** | 90° | 1050 步 | 60% | 中开 |
| **4 档** | 120° | 1400 步 | 80% | 大开 |
| **5 档** | 150° | 1750 步 | 100% | 全开 |

### 角度计算原理
- **零点定义**: 上电时的初始位置为零点（0 档）
- **相对角度**: 所有档位角度都相对于零点计算
- **步数计算**: `目标步数 = 档位 × 350 步`
- **往复运动**: 电机可在任意档位间来回转动

### 示例说明
```
场景 1：从 0 档到 3 档
- 当前位置：0 档（0 步）
- 目标位置：3 档（1050 步）
- 移动方向：顺时针
- 移动步数：1050 步

场景 2：从 5 档到 2 档
- 当前位置：5 档（1750 步）
- 目标位置：2 档（700 步）
- 移动方向：逆时针
- 移动步数：1050 步（回转）
```

## 🌡️ 温差自动调节

### 温差与档位映射关系

系统根据**实际温度与目标温度的差值**自动决定阀门开度：

| 温差范围（℃） | 目标档位 | 相对角度 | 阀门状态 |
|--------------|---------|---------|---------|
| ≤ -6.5 | 5 档 | 150° | 阀门大开（急需升温） |
| -6.5 ~ -3.5 | 4 档 | 120° | 阀门大开（快速升温） |
| -3.5 ~ -0.5 | 3 档 | 90° | 阀门中开（正常升温） |
| **-0.5 ~ +0.5** | **保持** | **-** | **【死区】停止调节** |
| +0.5 ~ +3.5 | 2 档 | 60° | 阀门中小开（缓慢升温） |
| +3.5 ~ +6.5 | 1 档 | 30° | 阀门小开（接近目标） |
| > +6.5 | 0 档 | 0° | 阀门关闭（温度过高） |

### 死区机制
- **死区范围**: ±0.5°C
- **作用**: 防止温度在目标值附近波动时阀门频繁动作
- **行为**: 进入死区后，电机立即停止，保持当前位置
## 🎮 控制模式

### 1. 自动模式
- **触发条件**: `valve_mode_auto = 1`
- **控制逻辑**: 根据温差自动调节档位
- **响应时机**:
  - 预设温度变化时立即响应
  - 每 15 秒周期性检测
- **用户干预**: 不可手动调节档位

### 2. 手动模式
- **触发条件**: `valve_mode_auto = 0`
- **控制逻辑**: 用户通过菜单直接设置档位
- **调节方式**: K3/K4 增减档位，K2 确认
- **自动保护**: 无自动调节

### 模式切换
- 在设置页（Page2）编辑 Mode 项
- K3/K4 切换 Auto/Manual
- K2 确认保存
