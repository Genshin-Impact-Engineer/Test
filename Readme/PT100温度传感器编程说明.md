# PT100 温度传感器模块 — AI 编程说明

## 基本参数

| 参数 | 值 |
|------|-----|
| 工作电压 | **DC 5.0V** |
| 输出信号 | **模拟电压 0~3.7V** |
| 测量范围 | **0~200°C** |
| 传感器类型 | PT100 RTD（铂电阻温度检测器） |
| 模块尺寸 | 35×14×1.6mm |
| 配套传感器 | PT100 探头 |

## 引脚定义

| 引脚 | 说明 |
|------|------|
| VCC | 接 DC 5.0V 电源正极 |
| GND | 接电源负极 |
| AO | 模拟信号输出 → 接 MCU ADC 引脚 |

## 温度 - 电压关系

### 近似公式（模块线性化后）

```
温度 (°C) = 电压值 (V) × 51.2
或
温度 (°C) = ADC 电压值 × 0.512  // 当电压以 0.01V 为单位时
```

### 示例代码中的转换

```c
// 代码来自官方测试程序
ad1 = Get_Adc_Average(1, 20);  // ADC 采集 20 次取平均
v1 = ad1 / 12.41;              // AD 值转电压值（放大 100 倍后的电压，单位 0.01V）
temp = v1 * 0.512;             // 电压值转温度值
```

### 完整计算公式

```
// 假设 ADC 为 12 位 (0~4095)，参考电压 3.3V
float adc_val = Get_Adc_Average(channel, 20);  // 读取 ADC
float voltage = (adc_val / 4096.0) * 3.3;      // 转换为电压值 (V)
float temperature = voltage * 51.2;            // 转换为温度 (°C)
```

## STM32 参考代码

```c
#include "adc.h"

// 初始化 ADC
Adc_Init();

// 读取温度
float read_pt100_temperature(void) {
    // 读取 ADC 值（假设使用 ADC1 通道 1）
    uint16_t adc_val = Get_Adc_Average(ADC_Channel_1, 20);
    
    // 转换为电压值 (假设 12 位 ADC, 3.3V 参考)
    float voltage = (float)adc_val / 4096.0f * 3.3f;
    
    // 转换为温度
    float temperature = voltage * 51.2f;
    
    return temperature;
}

// 主循环示例
while(1) {
    float temp = read_pt100_temperature();
    printf("Temperature: %.2f C\n", temp);
    delay_ms(300);
}
```

## 精度校准

PT100 传感器可以通过校准进一步提高精度：

1. 使用已知温度的参考源（如冰水混合物 0°C）
2. 测量此时模块输出电压
3. 计算偏差值并在校准公式中修正

```c
// 校准示例
#define TEMP_OFFSET  -2.5  // 校准偏移量（通过实测得到）

float read_calibrated_temperature(void) {
    float raw_temp = read_pt100_temperature();
    return raw_temp + TEMP_OFFSET;
}
```

## 注意事项

1. 模块供电必须为 **5.0V DC**，不可使用 3.3V
2. AO 输出电压范围 **0~3.7V**，可直接接 3.3V 系统的 MCU ADC
3. PT100 探头与模块之间通过导线连接，注意接线牢固
4. 长时间使用建议定期校准以保证测量精度
5. 模块已做线性化处理，可直接使用简化公式计算温度
