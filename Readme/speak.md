# 开发日志

## 2026-05-05 本轮修正（开机动画 + 语音完善）

### 一、修复上电黑屏

**问题**：上电/复位后 OLED 黑屏数秒（Sensor_Init 阻塞 ~2.5s 期间无显示）。

**修复**：重构 `Boot_Interface_Show()` 为三阶段：
1. 标题"智能生鲜结算系统"立即显示（用户看到画面）
2. `Sensor_Init()` 在标题保持显示期间执行
3. 进度条 + "Ready!"

传感器初始化不再造成黑屏感知。

### 二、ASR-PRO 语音指令完善

**新增指令**（对齐 [ASR-PRO.md](ASR-PRO.md)）：

| 指令 | 触发时机 | 按键 |
|------|---------|------|
| `Measure Complete` | 称量完成（结算） | K2 未选中 |
| `Tare` | 去皮 | K1 未选中 + 去皮前 |
| `Tare Off` | 取消去皮 | K1 未选中 + 已去皮 |
| `completed` | 更新完成 | K2 编辑保存 |
| `cancelled` | 取消更新 | K1 编辑取消 |
| `Overweight` | 超重报警 | 自动 |
| `Weight abnormal` | 重量异常报警 | 自动 |

**移除**：预览页 K2/K1 的语音触发（语音仅在称重页触发）。

**报警间隔**：恢复为 3s（对齐 ASR-PRO.md 第 18 行）。

### 三、字库扩展
