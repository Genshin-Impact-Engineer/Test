# OLED 菜单系统界面设计文档

## 硬件环境
- **屏幕**: OLED 显示屏
- **分辨率**: 128x64 像素
- **字体**: 8x16 字符（全屏可显示 4 行，每行约 16 字符）

---

## 开机界面（Boot Interface）

### 功能说明
- 系统上电后首先显示开机界面，执行一次后自动进入主菜单
- 总耗时约 2 秒（1 秒加载动画 + 1 秒 Ready 提示）
- 函数名：`Boot_Interface_Show(void)`

### 界面布局
```
┌────────────────────────────────┐
│                                │
│   Heating Valve Controller     │  ← 标题（第 2 行）
│                                │
│         Loading...             │  ← 加载提示
│   ▓▓▓▓▓▓▓░░░░░░░░░░░░░░   │  ← 进度条动画
│                                │
└────────────────────────────────┘
```

### 显示流程

| 步骤 | 内容 | 位置 | 说明 |
|------|------|------|------|
| 1 | 清屏初始化 | - | OLED 清屏，延时 100ms |
| 2 | 外边框 | 全屏 | 128x64 矩形边框 |
| 3 | 内边框（加载区） | (10,48)-(117,58) | 进度条外框 |
| 4 | 标题文字 | (0,12) | "Heating Valve Controller" |
| 5 | 加载提示 | (12,34) | "Loading..." |
| 6 | 进度条动画 | (11,49) 开始 | 从左到右渐变填充（0-107 像素） |
| 7 | 加载完成 | (40,33) | 显示"Ready!"，保持 1 秒 |
| 8 | 清屏 | - | 进入主菜单界面 |

### 进度条动画规格
- **起始位置**: (11, 49)
- **最大宽度**: 107 像素
- **填充方式**: 实心矩形渐变填充
- **动画速度**: 每步延时 10ms，共 108 步
- **总耗时**: 约 1 秒
- **刷新方式**: 每绘制一步立即刷新显示区域

### 加载完成提示
- **显示内容**: "Ready!"
- **显示位置**: (40, 33)
- **保持时间**: 1 秒
- **后续动作**: 清屏后进入主菜单

### 调用时机
- 在 `main()` 函数中，OLED 初始化之后调用
- 在其他外设初始化之前执行
- 仅执行一次，无循环

## 按键系统

| 按键 | 短按功能 | 长按功能 (≥1 秒) |
|------|----------|------------------|
| **K1** | 取消 / 返回 / 退出编辑 | - |
| **K2** | 进入编辑 / 确认修改 | - |
| **K3** | 向下选择 / 减少值 | 切换上一页 |
| **K4** | 向上选择 / 增加值 | 切换下一页 |

---

## 页面 1：数据页（主显示页）

### 布局
```
┌────────────────────────────────┐
│ Turb:XX.X NTU                  │  ← 第 1 行 (浊度实时值)
│ Temp:XX.X C                    │  ← 第 2 行 (温度实时值)
│ SetT:XX.X C                    │  ← 第 3 行 (预设目标温度)
│ Setting:X                      │  ← 第 4 行 (阀门档位 0-7)
└────────────────────────────────┘
```

### 显示内容

| 行 | 标签 | 说明 | 单位 | 可编辑 |
|---|---|---|---|---|
| 1 | Turb | 实时浊度值 | NTU | 否 |
| 2 | Temp | 实时温度值 | °C | 否 |
| 3 | SetT | 预设目标温度 | °C | 是 |
| 4 | Setting | 阀门档位 (0-7) | - | 是 (仅手动模式) |

### 交互逻辑

1. **浏览选择**
   - K4 短按：向上选择（SetT → Setting → SetT 循环）
   - K3 短按：向下选择（Setting → SetT → Setting 循环）
   - 选中项用矩形框高亮显示

2. **编辑模式**（K2 进入）
   - 选中项开始闪烁（300ms 间隔）
   - K4：增加数值（向上调节）
   - K3：减少数值（向下调节）
     - SetT: 步长 0.5°C，范围 16-40°C
     - Setting: 步长 1 档，范围 0-7 档
   - K2：确认保存
   - K1：取消退出

3. **自动刷新**
   - 每 500ms 刷新一次显示（非编辑模式下）

4. **阀门模式说明**
   - **自动模式**: Setting 显示根据温差自动计算的目标档位，不可编辑
   - **手动模式**: Setting 可编辑，用户直接设置阀门开度

## 页面 2：设置页（参数配置页）

### 布局
```
┌────────────────────────────────┐
│ MaxT:XX.X C                    │  ← 第 1 行 (温度报警上限)
│ MinT:XX.X C                    │  ← 第 2 行 (温度报警下限)
│ Turb:XX.X NTU                  │  ← 第 3 行 (浊度报警阈值)
│ Mode:Auto / Manual             │  ← 第 4 行 (阀门控制模式)
└────────────────────────────────┘
```

### 显示内容

| 行 | 标签 | 说明 | 默认值 | 可编辑 |
|---|---|---|---|---|
| 1 | MaxT | 温度报警上限 | 32.0°C | 是 |
| 2 | MinT | 温度报警下限 | 16.0°C | 是 |
| 3 | Turb | 浊度报警阈值 | 8.0 NTU | 是 |
| 4 | Mode | 阀门模式：Auto/Manual | - | 是 |

### 可调节参数及步长

| 参数 | 变量名 | 步长 | 范围 |
|------|--------|------|------|
| 温度上限 | temp_max | 0.5°C | 16.0-50.0°C |
| 温度下限 | temp_min | 0.5°C | 0.0-32.0°C |
| 浊度阈值 | turb_threshold | 0.5 NTU | 0.0-20.0 NTU |
| 阀门模式 | valve_mode_auto | 切换 | Auto/Manual |

### 交互逻辑

1. **浏览选择**
   - K4 短按：向上选择（Mode → Turb → MinT → MaxT → Mode 循环）
   - K3 短按：向下选择（MaxT → MinT → Turb → Mode → MaxT 循环）
   - 选中项用矩形框高亮显示

2. **编辑模式**（K2 进入）
   - 选中项开始闪烁（300ms 间隔）
   - K4：增加数值或切换选项（向上调节）
   - K3：减少数值或切换选项（向下调节）
     - MaxT: 步长 0.5°C
     - MinT: 步长 0.5°C
     - Turb: 步长 0.5 NTU
     - Mode: 切换 Auto ↔ Manual
   - K2：确认保存
   - K1：取消退出

## 页面 3：报警页（自动弹出）

### 布局（正常状态）
```
┌────────────────────────────────┐
│ Temp:OK!                       │
│                                │
│ Water:OK!                      │
│                                │
└────────────────────────────────┘
```

### 布局（温度异常）
```
┌────────────────────────────────┐
│ Temp:ERROR!                    │
│ Please check                   │
│ Water:OK!                      │
│                                │
└────────────────────────────────┘
```

### 布局（浊度异常）
```
┌────────────────────────────────┐
│ Temp:OK!                       │
│                                │
│ Water:ERROR!                   │
│ Need handle                    │
│                                │
└────────────────────────────────┘
```

### 布局（双重异常）
```
┌────────────────────────────────┐
│ Temp:ERROR!                    │
│ Please check                   │
│ Water:ERROR!                   │
│ Need handle                    │
└────────────────────────────────┘
```

### 触发条件

| 类型 | 条件 | 延迟 |
|------|------|------|
| 温度报警 | Temp < MinT 或 Temp > MaxT | 持续 5 秒 |
| 浊度报警 | Turb ≥ 阈值 | 持续 5 秒 |

### 报警行为

1. **自动跳转**: 满足任一条件自动切换到页面 3
2. **语音提示**: 通过 USART2 串口发送
   - 仅温度异常 → "temp"
   - 仅浊度异常 → "water"
   - 双重异常 → "temp and water"
3. **冷却机制**: 第一次报警后，按下任意按键返回报警前的页面，进入 60 秒冷却期，不重复报警
4. **重新计时**: 异常消除后再次异常，重新 5 秒计时

### 交互逻辑

- **任意按键**: 第一次报警后，按下任意按键返回报警前的页面（页面 1 或页面 2）

## 页面切换逻辑

### 循环顺序
```
数据页 (Page1) → 设置页 (Page2) → 报警页 (Page3) → 数据页 (Page1) → ...
```

### 切换方式

| 操作 | 方向 | 说明 |
|------|------|------|
| 长按 K4 ≥1 秒 | 下一页 | 正向循环 |
| 长按 K3 ≥1 秒 | 上一页 | 反向循环 |

### 切换行为
- 切换页面时自动退出编辑模式
- 选中项重置为 0

## 全局变量（供参考）

| 变量 | 类型 | 说明 |
|------|------|------|
| current_page | uint8_t | 当前页面 ID |
| selected_item | uint8_t | 当前选中项 (0=无，1=Item1, 2=Item2) |
| edit_mode | uint8_t | 编辑模式标志 |
| temp_max | float | 温度报警上限（默认 32.0°C） |
| temp_min | float | 温度报警下限（默认 16.0°C） |
| turb_threshold | float | 浊度报警阈值（默认 8.0 NTU） |
| preset_temp | float | 预设目标温度 |
| water_valve_level | uint8_t | 阀门档位 (0-7) |
| valve_mode_auto | uint8_t | 阀门模式 (1=自动，0=手动) |

| 元素 | 说明 |
|------|------|
