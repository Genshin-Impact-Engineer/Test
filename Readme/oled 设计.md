# OLED 菜单系统界面设计文档

## 硬件环境
- **屏幕**: OLED 显示屏
- **分辨率**: 128x64 像素
- **字体**: 8x16 字符（全屏可显示 4 行，每行约 16 字符）

---

## 按键系统

| 按键 | 短按功能 | 长按功能 (≥1 秒) |
|------|----------|------------------|
| **K1** | 取消 / 返回 / 退出编辑 | - |
| **K2** | 进入编辑 / 确认修改 | - |
| **K3** | 向上选择 / 减少值 | 切换上一页 |
| **K4** | 向下选择 / 增加值 | 切换下一页 |

---

## 页面 1：数据页（主显示页）

### 布局
```
┌────────────────────────────────┐
│ Turb:XX.X NTU                  │  ← 第 1 行 (浊度实时值)
│ Temp:XX.X C                    │  ← 第 2 行 (温度实时值)
│ SetT:XX.X C                    │  ← 第 3 行 (预设目标温度)
│ Setting:X                      │  ← 第 4 行 (阀门档位 0-5)
└────────────────────────────────┘
```

### 显示内容

| 行 | 标签 | 说明 | 单位 | 可编辑 |
|---|---|---|---|---|
| 1 | Turb | 实时浊度值 | NTU | 否 |
| 2 | Temp | 实时温度值 | °C | 否 |
| 3 | SetT | 预设目标温度 | °C | 是 |
| 4 | Setting | 阀门档位 (0-5) | - | 是 (仅手动模式) |

### 交互逻辑

1. **浏览选择**
   - K3/K4 短按：循环选择 SetT 或 Setting 项
   - 选中项用矩形框高亮显示

2. **编辑模式**（K2 进入）
   - 选中项开始闪烁（300ms 间隔）
   - K3/K4：增减数值
     - SetT: 步长 0.5°C，范围 0-50°C
     - Setting: 步长 1，范围 0-5
   - K2：确认保存
   - K1：取消退出

3. **自动刷新**
   - 每 500ms 刷新一次显示（非编辑模式下）

4. **阀门模式说明**
   - **自动模式**: Setting 显示根据温差自动计算的目标档位，不可编辑
   - **手动模式**: Setting 可编辑，用户直接设置阀门开度

---

## 页面 2：设置页（参数配置页）

### 布局
```
┌────────────────────────────────┐
│ MaxT:XX.X C                    │  ← 第 1 行 (温度报警上限)
│ MinT:XX.X C                    │  ← 第 2 行 (温度报警下限)
│ Turb:XX.X NTU                  │  ← 第 3 行 (浊度报警阈值)
│ Mode:Auto / Manual             │  ← 第 4 行 (阀门控制模式)
└────────────────────────────────┘
```

### 显示内容

| 行 | 标签 | 说明 | 默认值 | 可编辑 |
|---|---|---|---|---|
| 1 | MaxT | 温度报警上限 | 可配置 | 否 |
| 2 | MinT | 温度报警下限 | 可配置 | 否 |
| 3 | Turb | 浊度报警阈值 | 可配置 | 否 |
| 4 | Mode | 阀门模式：Auto/Manual | - | 是 |

### 交互逻辑

1. **浏览选择**
   - K3/K4 短按：选择 Mode 行（矩形框高亮）

2. **编辑模式**（K2 进入）
   - Mode 项开始闪烁
   - K3/K4：切换 Auto ↔ Manual
   - K2：确认保存
   - K1：取消退出

## 页面 3：报警页（自动弹出）

### 布局（正常状态）
```
┌────────────────────────────────┐
│ Temp:OK!                       │
│                                │
│ Water:OK!                      │
│                                │
└────────────────────────────────┘
```

### 布局（温度异常）
```
┌────────────────────────────────┐
│ Temp:ERROR!                    │
│ Please check                   │
│ Water:OK!                      │
│                                │
└────────────────────────────────┘
```

### 布局（浊度异常）
```
┌────────────────────────────────┐
│ Temp:OK!                       │
│                                │
│ Water:ERROR!                   │
│ Need handle                    │
│                                │
└────────────────────────────────┘
```

### 布局（双重异常）
```
┌────────────────────────────────┐
│ Temp:ERROR!                    │
│ Please check                   │
│ Water:ERROR!                   │
│ Need handle                    │
└────────────────────────────────┘
```

### 触发条件

| 类型 | 条件 | 延迟 |
|------|------|------|
| 温度报警 | Temp < MinT 或 Temp > MaxT | 持续 5 秒 |
| 浊度报警 | Turb ≥ 阈值 | 持续 5 秒 |

### 报警行为

1. **自动跳转**: 满足任一条件自动切换到页面 3
2. **语音提示**: 通过 USART2 串口发送
   - 仅温度异常 → `"temp"`
   - 仅浊度异常 → `"water"`
   - 双重异常 → `"temp and water"`
3. **冷却机制**: K1 确认后进入 60 秒冷却期，不重复报警
4. **重新计时**: 异常消除后再次异常，重新 5 秒计时

### 交互逻辑

- **K1 短按**: 返回报警前的页面（页面 1 或页面 2）

---

## 页面切换逻辑

### 循环顺序
```
数据页 (Page1) → 设置页 (Page2) → 报警页 (Page3) → 数据页 (Page1) → ...
```

### 切换方式

| 操作 | 方向 | 说明 |
|------|------|------|
| 长按 K4 ≥1 秒 | 下一页 | 正向循环 |
| 长按 K3 ≥1 秒 | 上一页 | 反向循环 |

### 切换行为
- 切换页面时自动退出编辑模式
- 选中项重置为 0

## 全局变量（供参考）

| 变量 | 类型 | 说明 |
|------|------|------|
| current_page | uint8_t | 当前页面 ID |
| selected_item | uint8_t | 当前选中项 (0=无，1=Item1, 2=Item2) |
| edit_mode | uint8_t | 编辑模式标志 |
| temp_max | float | 温度报警上限 |
| temp_min | float | 温度报警下限 |
| turb_threshold | float | 浊度报警阈值 |
| preset_temp | float | 预设目标温度 |
| water_valve_level | uint8_t | 阀门档位 (0-5) |
| valve_mode_auto | uint8_t | 阀门模式 (1=自动，0=手动) |

## 视觉元素规范

| 元素 | 说明 |
|------|------|
| 矩形框 | 选中项周围绘制空心矩形边框 |
| 闪烁 | 编辑模式下选中项每 300ms 切换显示/空白 |
| 刷新 | 数据页和报警页每 500ms 自动刷新 |
