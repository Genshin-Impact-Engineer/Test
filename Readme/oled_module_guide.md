# OLED 显示模块指南（电子秤/POS 系统）

## 一、硬件规格

| 参数 | 规格 |
|------|------|
| 屏幕类型 | SSD1306 OLED |
| 分辨率 | 128 × 64 像素 |
| 接口 | I2C（I2C1, SCL=PB8, SDA=PB9）|
| 从机地址 | 0x78（7-bit） |
| 字体 | 8×16 像素 ASCII，4 行 × 16 列 |
| 帧缓冲 | 1024 字节（128×64 / 8） |

## 二、界面总览

三页面系统，循环切换：

```
称重页 (PAGE_WEIGHING)  ←→  预览编辑页 (PAGE_PREVIEW)
        ↑                           │
        └──────── 报警页 ───────────┘
              (PAGE_ALARM)
```

**注意**：本系统**不使用长按翻页**。页面切换由按键逻辑控制：
- 称重 → 预览：KEY2 结算触发
- 预览 → 称重：KEY2 保存或 KEY3 取消
- 报警触发：重量异常自动跳入
- 报警退出：任意按键返回称重页

## 三、Page_one：称重页

### 布局

```
┌────────────────────────────────┐
│ Category:  Apple               │  ← 第 0 行（可选中，item=1）
│ UniPrice:  5.0Y/kg             │  ← 第 1 行（可选中，item=2）
│ Weight:    1.250 kg            │  ← 第 2 行（只读，实时刷新）
│ Total:     6.25 Y              │  ← 第 3 行（只读，自动计算）
└────────────────────────────────┘
```

### 可操作项

| 操作 | 按键 | 条件 | 效果 |
|------|------|------|------|
| 光标上移 | KEY0 | 未编辑 | 0 → 类别 → 单价 → 0（循环）|
| 光标下移 | KEY1 | 未编辑 | 0 → 单价 → 类别 → 0（循环）|
| 结算 | KEY2 | 未选中 | 计算总价，进入预览页 |
| 进入编辑 | KEY2 | 选中类别/单价 | 进入编辑模式（闪烁） |
| 重称 | KEY3 | 未选中 | 触发重量重新采集 |
| 退出选中 | KEY3 | 选中类别/单价 | 取消选中 |
| 退出编辑 | KEY0/KEY3 | 编辑模式 | 退出编辑并取消选中 |
| 下一商品 | KEY1 | 编辑类别 | 切换到下一个商品（单价自动更新） |
| 单价-0.5 | KEY1 | 编辑单价 | 单价减 0.5，下限 0.5 |
| 确认 | KEY2 | 编辑模式 | 退出编辑 |

## 四、Page_two：预览编辑页

### 布局

```
┌────────────────────────────────┐
│ [Preview]                      │  ← 第 0 行（状态提示）
│ Category:  Apple               │  ← 第 1 行（可编辑，item=1）
│ UniPrice:  5.0Y/kg             │  ← 第 2 行（可编辑，item=2）
│ Total:     6.25 Y              │  ← 第 3 行（预览总价）
└────────────────────────────────┘
```

### 可操作项

| 操作 | 按键 | 条件 | 效果 |
|------|------|------|------|
| 光标上移 | KEY0 | 未编辑 | 0 → 类别 → 单价 → 0 |
| 光标下移 | KEY1 | 未编辑 | 0 → 单价 → 类别 → 0 |
| 保存返回 | KEY2 | 任意 | 保存修改，返回称重页 |
| 放弃返回 | KEY3 | 任意 | 放弃修改，返回称重页 |
| 下一商品 | KEY1 | 编辑类别 | 切换商品（自动更新单价） |
| 单价-0.5 | KEY1 | 编辑单价 | 单价减 0.5 |
| 退出选中 | KEY0 | 编辑模式 | 退出但保持在预览页 |

## 五、Page_three：报警页

### 布局（超重）

```
┌────────────────────────────────┐
│ !!   WARNING   !!              │  ← 第 0 行
│ Overweight!                    │  ← 第 1 行
│ Remove items                   │  ← 第 2 行
│ Press any key...               │  ← 第 3 行
└────────────────────────────────┘
```

### 布局（重量异常）

```
┌────────────────────────────────┐
│ !!   WARNING   !!              │
│ Weight Error!                  │
│ Please re-weigh                │
│ Press any key...               │
└────────────────────────────────┘
```

### 触发条件

| 类型 | 条件 | 阈值 |
|------|------|------|
| 超重 | 重量 > 30.0 kg | WEIGHT_MAX |
| 重量异常 | 重量 < 0.005 kg（传感器故障/空载异常） | WEIGHT_MIN |

### 退出方式

**任意按键**（KEY0/KEY1/KEY2/KEY3）退出报警页面，返回称重页。同时：
- 清除报警标志
- 停止蜂鸣器
- 重置重量为 0

## 六、代码架构

```
oled.h / oled.c
├── OLED_Setup()             → OLED 初始化 + 默认参数
├── Boot_Interface_Show()    → 开机动画（进度条）
├── OLED_UpdateDisplay()     → 主刷新循环（渲染 + I2C 故障恢复）
├── OLED_HandleKey()         → 三页面按键分发
├── OLED_SetAlarmFlags()     → 设置报警标志
├── OLED_SetCategoryByName() → 蓝牙命令：设置类别
├── OLED_SetUnitPrice()      → 蓝牙命令：设置单价
└── Scale_TriggerReweigh()   → 触发重称
```

## 七、I2C 故障恢复

STM32F1 I2C 外设存在 BUSY 锁死缺陷。本模块内置双重恢复机制：

1. **每次刷新前主动探测**: 发送 `ENTIRE_DISPLAY_OFF` 命令，超时 5ms
2. **定期强制恢复**: 每 200 次刷新（约 100 秒）执行一次 I2C 外设复位
3. **Flush 失败即时恢复**: `OLED_Flush()` 返回非 OK 时立即恢复并重试

恢复步骤：
```
FORCE_RESET I2C1 → 延时 1ms → RELEASE_RESET → MX_I2C1_Init() → OLED_Init()
```

## 八、视觉效果

| 效果 | 说明 |
|------|------|
| 选中框 | 128×16 空心矩形框包围当前选中行 |
| 闪烁 | 编辑模式下，选中行每 300ms 切换显示/空白 |
| 刷新率 | 正常模式 500ms，编辑模式 300ms（闪烁同步） |

## 九、商品表

商品表定义在 `product_table[PRODUCT_COUNT]`，每个商品包含名称和默认单价：

```c
const Product_t product_table[PRODUCT_COUNT] = {
    {"Apple",  5.0f},
    {"Banana", 3.0f},
    // ... 最多 8 种商品
};
```

**扩展方法**：修改 `PRODUCT_COUNT` 宏并添加表项即可。
